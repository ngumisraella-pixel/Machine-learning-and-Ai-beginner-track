# Common Errors and Fixes

## AttributeError
- **Cause**: Accessing non-existent attribute
- **Fix**: Check spelling, use hasattr() first

## TypeError
- **Cause**: Wrong type for operation
- **Fix**: Validate input types, use isinstance()

## NameError
- **Cause**: Variable not defined
- **Fix**: Check scope, import modules

## IndentationError
- **Cause**: Incorrect indentation
- **Fix**: Use consistent indentation

## RecursionError
- **Cause**: Stack overflow from deep recursion
- **Fix**: Use iteration, add base case

## Memory Issues
- **Cause**: Large data structures
- **Fix**: Use generators, delete unused objects
