# Part-0 Concept Quizzes

## Variables & Types Quiz
1. What does `type(5.0)` return?
2. Convert the string "42" to an integer and add 8 to it
3. What's the difference between `=` and `==`?
4. True or False: You can reassign a variable's type

**Answers:** 1. `<class 'float'>` 2. `int("42") + 8 = 50` 3. `=` assigns, `==` compares 4. True

## Strings Quiz
1. Get the 3rd character from "Python"
2. Make "HELLO" lowercase
3. Split "one,two,three" by comma
4. How do you reverse a string?

**Answers:** 1. `"Python"[2] = 't'` 2. `"HELLO".lower() = "hello"` 3. `"one,two,three".split(',')` 4. `string[::-1]`

## Lists Quiz
1. Create a list with 3 items
2. Add an item to the end
3. Get the last item without knowing length
4. Loop through list with index

**Answers:** 1. `[1, 2, 3]` 2. `.append(item)` 3. `lst[-1]` 4. `for i, item in enumerate(lst):`

## Dictionaries Quiz
1. Create dict with name "Bob" and age 30
2. Get value for key "name"
3. Get all keys safely
4. True/False: Dict keys must be unique

**Answers:** 1. `{"name": "Bob", "age": 30}` 2. `d["name"]` 3. `d.keys()` 4. True
