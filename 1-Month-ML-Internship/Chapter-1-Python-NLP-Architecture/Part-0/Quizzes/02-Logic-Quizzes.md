# Logic & Loops Quizzes

## Conditionals Quiz
1. True: `5 > 3 and 2 < 4`?
2. What does `not True` equal?
3. Write if/else for "age >= 18"
4. What's the order: if, elif, else?

**Answers:** 1. Yes 2. False 3. Works as expected 4. if → elif → else

## Loops Quiz  
1. Loop 0 to 4: which range?
2. Break means: stop/skip/restart?
3. Continue means: skip/stop/restart?
4. How to get index in for loop?

**Answers:** 1. `range(5)` 2. Stop 3. Skip to next 4. `enumerate()`

## Functions Quiz
1. How many `return` statements allowed?
2. What's the name of `def add(a, b=5)` - b?
3. Can function return multiple values?
4. What if no return statement?

**Answers:** 1. Any number 2. Default parameter 3. Yes (as tuple) 4. Returns None
