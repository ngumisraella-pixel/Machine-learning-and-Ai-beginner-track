# Concept Map - Part 0

How everything connects!

## Foundation Layer
- Variables & Types (int, float, str, bool)
  ├── Type Checking (type())
  └── Type Conversion (int(), float(), str())

## Sequence Layer
- Strings
  ├── Indexing & Slicing
  ├── Methods (lower, upper, split, replace)
  └── Formatting (f-strings)

- Lists
  ├── Indexing & Slicing
  ├── Mutation (append, insert, remove)
  └── Iteration (for, enumerate)

- Tuples (immutable version of lists)
- Sets (unique collections)

## Mapping Layer
- Dictionaries
  ├── Key-Value Access
  ├── Methods (keys, values, items)
  └── Iteration

## Logic Layer
- Conditionals (if/elif/else)
  ├── Comparison Operators
  └── Logical Operators (and, or, not)

- Loops (for, while)
  ├── Break & Continue
  └── Nested Loops

## Organization Layer
- Functions
  ├── Parameters & Arguments
  ├── Return Values
  └── Scope (local vs global)

- Modules (import)

## Advanced Layer
- List Comprehensions
  └── Filtering & Transformation

- Lambda Functions
  └── With map() & filter()

## Scientific Layer
- NumPy
  ├── Arrays
  ├── Operations
  └── Statistical Functions
