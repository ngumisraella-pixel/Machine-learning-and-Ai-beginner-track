# Project Guidelines for Students

## What Each Project Teaches

### Project 1: Text Analyzer
**Skills:** Strings, lists, dictionaries, functions, file I/O
**Real-world use:** Processing text data for NLP
**Difficulty:** Intermediate
**Time:** 4-6 hours

### Project 2: Grade Manager
**Skills:** Dictionaries, lists, statistics, file I/O
**Real-world use:** Data storage and retrieval
**Difficulty:** Intermediate
**Time:** 3-4 hours

### Project 3: Data Normalizer
**Skills:** Loops, comprehensions, math, error handling
**Real-world use:** Feature scaling in ML
**Difficulty:** Advanced
**Time:** 5-7 hours

### Project 4: Final Project
**Skills:** All of Part-0
**Real-world use:** End-to-end data pipeline
**Difficulty:** Advanced
**Time:** 8-10 hours

## How to Use Projects
1. **Read project requirements** completely first
2. **Plan your approach** before coding
3. **Build incrementally** - test each part
4. **Compare with solution** - but try first!
5. **Extend it** - add your own features

## Project Submission Checklist
- [ ] Code runs without errors
- [ ] Code is readable with comments
- [ ] All requirements met
- [ ] Tested with edge cases
- [ ] Can explain every line
